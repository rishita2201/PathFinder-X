# Path-Finder X  
## AI-Based Skill Gap Analysis & Career Recommendation Engine

Path-Finder X is an AI-driven system that analyzes student resumes using NLP and Machine Learning to identify skill gaps and recommend suitable career paths along with a personalized learning roadmap.

---

## Problem Statement
Students often lack awareness of industry-required skills for specific career paths. This system bridges the gap between academic learning and industry expectations using AI-based resume analysis.

---

## Features
- Resume (PDF) analysis
- NLP-based skill extraction
- Machine Learning career prediction
- Soft skill detection
- Skill gap identification
- Learning roadmap generation
- Employability Index calculation
- Bias & ethics check
- Explainable AI outputs

---

## System Architecture

Resume → NLP Processing → ML Career Classifier → Skill Gap Analyzer → Roadmap Generator → Dashboard

---

## Tech Stack
- Python
- Flask
- Scikit-learn
- TF-IDF Vectorizer
- Naive Bayes Classifier
- HTML/CSS
